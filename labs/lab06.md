# Лабораторная работа №6. Основы web-разработки

Необходимо реализовать web-интерфейс для просмотра всех объектов, хранящихся в базе данных. Предметная область - из лабораторной работы 3 (4). Для доступа к данным желательно использовать ORM peewee (возможны альтернативные библиотеки). Для реализации web-части - cerrypy (если не знакомы с HTML, CSS, то можно сделать телеграм бота, который выводит данные из базы данных и позволяет их добавлять).

Добавление / изменение необходимо реализовать минимум для 1 таблицы БД с внешними ключами.
