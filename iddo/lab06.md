# Лабораторная работа №6. Основы web-разработки

Необходимо реализовать web-интерфейс для просмотра всех объектов, хранящихся в базе данных. Предметная область - из лабораторной работы 3 (4). Для доступа к данным желательно использовать ORM peewee (возможны альтернативные библиотеки). Для реализации web-части - cerrypy. Ajax использовать необязательно, можно сделать все на статических объектах.

Добавление / изменение необходимо реализовать минимум для 1 таблицы БД с внешними ключами.
