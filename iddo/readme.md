# Задание на курсовую работу
Необходимо разработать средство автоматизации по варианту, который задается в лабораторных работах 3,4,6
Для выбранной предметной области (по варианту) необходимо разработать средство автоматизации, удовлетворяющее следующим условиям:


* Программные средства в виде web-приложения для создания, редактирования, удаления и просмотра сущностей, заданных вариантом.
* СУБД на выбор (Postgres, MySQL, SQLite, MS SQL, Oracle, DB2, ...) - не менее 7 таблиц. 
* В отчете необходимо обосновать выбор структуры хранения данных. В структуре СУБД должны быть предусмотрены первичные ключи, автоинкрементные поля.
* Требования к клиентской части: для каждой операции / сущности должна быть своя отдельная страница. Связанные сущности можно редактировать на одной и той же странице.
* Требования к серверной части: она должна быть написана на языке python с использованием любого фреймворка: cherrypy, flask, django.
